Welcome to Berkeley High’s Intro to Programming class.

This website consists of a number of tools you will use to do some of your first assignments for this class. We’ll go over getting set up in the first day or two of class but the main thing you need to do is sign up for a free GitHub account if you don’t already have one. (If you already have a GitHub account, it’s probably easiest to use that one for this class but if you’d rather create a new account for class that’s fine too.)

If you've set up your GitHub account you can look at the tools here. Or if you haven't you can still play with them but you won't be able to save your work.
